StageLayout Designer
📌 概要
StageLayout Designer は、音響・照明・舞台・大道具を含むイベント全体のレイアウトを直感的に作成・保存・印刷できるツールです。
大学での音響経験と照明現場での経験をもとに、実際の仕込み図作成をより効率化することを目的として開発しています。

✨ 特徴
🎤 音響・照明・舞台・大道具・人物・会場をカテゴリ別に管理

🖱️ ドラッグ＆ドロップで機材を配置

↻ 右クリックで向き変更（90° / 180°）

🎨 色変更・サイズ変更に対応

🧍 棒人間アイコンで立ち位置を表現

🟦 バミリ（立ち位置テープ）を配置可能

🖨️ PDF 印刷対応（仕込み図＋機材一覧）

📏 グリッド（マス目）で実寸を把握しやすい
